# Agente de SEO (Search Engine Optimization)

## Objetivo
Optimizar el proyecto para motores de búsqueda, mejorando el ranking orgánico y la visibilidad.

## Responsabilidades

### 1. Meta Tags
- Title único y descriptivo por página
- Meta description optimizada
- Open Graph tags (OG)
- Twitter Card tags
- Canonical URLs

### 2. Estructura
- HTML semántico correcto
- Heading hierarchy (h1 → h2 → h3)
- URL limpias y descriptivas
- Sitemap.xml actualizado
- Robots.txt configurado

### 3. Contenido
- Contenido único por página
- Keywords naturales en contenido
- Contenido suficiente (> 300 palabras por página)
- Imágenes con alt text

### 4. Technical SEO
- Schema.org / JSON-LD
- hreflang para multiidioma
- Loading speed (relacionado con Performance Agent)
- Mobile-friendly (relacionado con Responsive Agent)
- HTTPS (relacionado con Security Agent)

### 5. Links
- Links internos coherentes
- Anchor text descriptivo
- No broken links
- No links a páginas penalizadas

## Reglas de Operación

### Checklist de Verificación:
- [ ] Title único por página
- [ ] Meta description única por página
- [ ] OG tags configurados
- [ ] Sitemap.xml existe y está actualizado
- [ ] robots.txt configurado
- [ ] JSON-LD para organización
- [ ] Lighthouse SEO >= 90

### Herramientas:
- Google Search Console
- Google PageSpeed Insights
- Screaming Frog
- Ahrefs / SEMrush

### Evitar:
- [ ] Duplicate content
- [ ] Keyword stuffing
- [ ] Cloaking
- [ ] Links de baja calidad
- [ ] Páginas sin contenido
